## 📝 文章模板

### 知鱼说模板

**开头**：
```markdown
> 🐟 知鱼说：别被"..."这种假象骗了，今天咱只讲一个事儿——[核心观点]
```

**章节**：
```markdown
🔥 **发生了什么？**
📊 **现状分析**
💡 **正确做法**
🎯 **问题根源**
⚡ **具体建议**
```

**结尾**：
```markdown
> 🐟 **关于知鱼**
>
> 专注 AI 赛道的实战派，不整虚的概念，只讲能落地的玩法。
>
> 关注我，一起在这个 AI 爆炸的时代，找到属于你的机会。
```

### 卡兹克模板

**开头（选择一种）**：
- "故事是这样的。"
- "最近这两天，被...给刷屏了。"
- "这两天在网上刷到了一张图，很有意思。"

**转场词**：
- 坦率的讲、说真的、怎么说呢、回到这块、顺着上面的再聊聊

**结尾**：
```markdown
以上，既然看到这里了，如果觉得不错，随手点个赞、在看、转发三连吧，如果想第一时间收到推送，也可以给我个星标⭐～

谢谢你看我的文章，我们，下次再见。

> / 作者：卡兹克
```

## 🗣️ 口语化词库

### 知鱼说常用

- 转场：坦率的讲、说实话、怎么说呢
- 表达：我觉得、我认为、我的理解是
- 自嘲：我也不确定、还在摸索

### 卡兹克常用

- 转场：这玩意、不是哥们、说真的、太牛逼了
- 情绪：我当时就愣住了、太离谱了、鬼使神差的
- 自嘲：愚钝如我、老阴逼、我寻思了一下

## ✅ 质量检查（所有风格统一）

### L1 硬性规则

| 检查项 | 要求 |
|--------|------|
| 禁用词 | ❌ 说白了、意味着什么、本质上、换句话说 |
| 禁用标点 | ❌ 冒号：、破折号——、双引号"" → 用「」 |
| 结构套话 | ❌ 让我们来看看、随着...的发展 |
| 工具名 | ✅ 全部具体名称 |

### L2 风格一致性

- [ ] 开头从具体事件切入
- [ ] 至少3处单句独立成段
- [ ] 口语化词组 ≥ 8个
- [ ] 至少1处自嘲/承认不足

### L3 内容质量

- [ ] 观点有场景/数据支撑
- [ ] 知识自然带出
- [ ] **有文化升维**（连接更大参照物）
- [ ] 有对立面理解

### L4 活人感

- [ ] 体感记忆式情绪
- [ ] 独特角度
- [ ] 非导师姿态
- [ ] 心流顺畅

## 🔍 素材采集

### 双引擎搜索（推荐）

```bash
py ai-wechat-publisher/scripts/research.py research "[主题]" --engine all --save output/research_notes.md
```

### Exa 语义搜索

```bash
py ai-wechat-publisher/scripts/research.py exa-search "[主题]" --category news --with-summary
```

```bash
py ai-wechat-publisher/scripts/research.py search "[主题]" --topic news --time-range week
```

| 引擎 | 擅长 | 命令 |
|------|------|------|
| Tavily | 综合、新闻 | `search` |
| Exa | 语义、学术 | `exa-search` |
